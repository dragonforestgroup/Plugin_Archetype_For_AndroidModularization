#Android组件化开发插件
## 功能： 
    帮助开发者快速创建组件化项目，快速对组件化模块进行配置，插件分两个功能，创建项目和创建模块

### 1.创建项目

*  1.自动创建对应包名的项目。
*  2.自动创建config.gradle配置文件和引入
*  3.自动划分业务包名
*  4.自动加入公司信息

### 2.创建模块
*  1.自动创建模块
*  2.自动配置build.gradle文件，包含默认sdk版本和组件化配置
*  3.自动将模块信息添加到settings.gradle文件
*  4.自动创建组件化清单libmanifest
*  5.自动添加组件化配置变量（添加在gradle.properties中）
